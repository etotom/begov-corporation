# Begov Corporation — Auto Export

Сайт компании по экспорту автомобилей из США, Европы и ОАЭ через Грузию в
Таджикистан, Узбекистан, Казахстан, Кыргызстан и Россию.

**Стек:** Next.js 16 (App Router) · TypeScript · Tailwind CSS 4 · Vercel

## Страницы

| Раздел | Путь | Описание |
| --- | --- | --- |
| Главная | `/` | Позиционирование, процесс, география, авто в наличии |
| Каталог | `/catalog` | Автомобили с фильтрами (регион, кузов, статус, сортировка) |
| VIN-проверка | `/vin` | Бесплатная расшифровка VIN через API NHTSA vPIC |
| Калькулятор | `/calculator` | Смета «под ключ» с разбивкой по статьям |
| Услуги | `/services` | Полный цикл: аукцион → доставка → автовоз → документы |
| О компании | `/about` | Инфраструктура и принципы |
| Контакты | `/contacts` | Форма заявки + контакты |
| Регистрация / Вход | `/auth/register`, `/auth/login` | Личный кабинет |
| Кабинет | `/account` | Профиль и история заявок |

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # продакшен-сборка
```

## Что заменить перед запуском в продакшен

- **Контакты** — телефон `+995 000 00 00 00`, email и адрес — плейсхолдеры.
  Файлы: `components/Footer.tsx`, `app/contacts/ContactsClient.tsx`.
- **Тарифы калькулятора** — ориентировочные ставки в `app/calculator/CalculatorClient.tsx`
  (объект `RATES`).
- **Каталог** — демо-данные в `lib/cars.ts`. Замените на реальные машины со стоянки
  (фото добавляются полем в этой же структуре).

## Дорожная карта (v2)

- **База данных** (Neon Postgres через Vercel Marketplace) + Auth.js: сейчас
  регистрация и заявки хранятся в браузере клиента (`lib/auth.ts`, `lib/leads.ts`) —
  интерфейсы написаны так, чтобы замена на серверную реализацию не трогала страницы.
- **Прием заявок**: `app/api/lead/route.ts` пишет в логи — подключить Telegram-бота
  или почту.
- **Админка каталога**: загрузка машин с фото вместо `lib/cars.ts`.
- **Языки**: узбекский / таджикский / казахский / английский (i18n).
- **Платная VIN-история** (Carfax/AutoCheck реселлер) как отдельная услуга.
